이후 단계별 목표 (일정의 상황 봐가며)

1 단계 : 기본 게시판

   목록 -> 목록검색 -> 상세 -> 수정 -> 삭제 -> 페이징

2 단계 : 회원모듈

  회원 가입 -> 로그인/아웃 -> 자기정보 상세 -> 수정 -> 탈퇴

3 단계 : 회원제 게시판

  1단계의 게시판 복사하여 로그인 해야 글 등록 / 자기글만 수정,삭제 기능 붙이기

4 단계 : 기본 관리자

  관리자 로그인/아웃

  회원 관리 : 회원목록 -> 검색 -> 수정 -> 삭제

5 단계 : 멀티형 게시판

  게시판 카테고리 관리 : 등록 -> 목록 -> 수정 -> 삭제

6 단계 : 멀티 게시판 적용

  메인에서 게시판 목록이 나오게 하고 클릭하면 해당 게시판 사용
